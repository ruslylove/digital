# Digital Circuit and Logic Design

**Department of Electrical and Computer Engineering (ECE)**
**Faculty of Engineering**
**King Mongkut's University of Technology North Bangkok (KMUTNB)**

This repository contains the lecture slides and course materials for the **Digital Circuit and Logic Design** course. The materials are built using [Slidev](https://github.com/slidevjs/slidev) for interactive and engaging presentations.

## Course Overview

This course covers the fundamental principles of digital systems, including:
*   Number systems and codes
*   Boolean algebra and logic gates
*   Combinational logic design (Decoders, Encoders, Multiplexers, Arithmetic circuits)
*   Sequential logic design
*   VHDL for digital system modeling

## Getting Started

To run the slides locally, ensure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

1.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

2.  **Start the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```

3.  **View the slides:**
    Open your browser and visit `http://localhost:3030`
